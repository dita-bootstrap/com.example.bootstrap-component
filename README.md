# DITA Bootstrap Component

This is a sample DITA-OT plug-in that creates a new `<callout>` specialization as described in the 
DITA Bootstrap plug-in documentation under [Custom Component](https://dita-bootstrap.github.io/component.html).

> [!IMPORTANT]  
> Requires the [dita-bootstrap.html](https://github.com/dita-bootstrap/dita-bootstrap.html) plug-in for DITA Open Toolkit.  
> For details, see [dita-bootstrap.github.io](https://dita-bootstrap.github.io/).
